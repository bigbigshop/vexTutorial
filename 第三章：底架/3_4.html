3.4 底盤結構與雙重支撐 (Parallel Rail)

前情提要​

坦克底盤 (3.1) 配合混合式輪胎配置 (3.3)，在理論上具備最強的防禦力。然而，若底盤結構鬆散，一旦受到撞擊便發生歪斜，再優良的輪胎配置也無法發揮作用。

本章將介紹如何構建具備高度穩定性的「三明治」底盤（Boxed Chassis / Parallel Rail），確保結構如鋼板般堅硬。

1. 常見結構缺陷：單片支撐​

新手最常犯的結構錯誤是採用 「單片支撐」 設計。

  • 設計方式： 輪軸僅穿過單片 C 型鋼（C-Channel）。
  • 結構後果： 這在力學上稱為懸臂軸 (Cantilevered Shaft)。當機器人受到側面撞擊（T-Bone）時，巨大的瞬間應力會集中於細長的輪軸上，導致軸心彎曲或 C 型鋼扭曲，最終令輪胎出現「外八字」偏移，大幅增加傳動阻力。

2. 核心方案：三明治底盤 (Parallel Rail Bracing)​

為了提升剛性，目前學界主流採用「三排孔樑結構」，亦即將輪子安置於兩片金屬樑之間。


🥪 結構圖解​

  • 內側樑： 一根長型的 C 型鋼。
  • 外側樑： 另一根平行的 C 型鋼。
  • 配置： 輪胎被夾在兩根樑之間，形成對稱支撐。
  • 貫穿軸： 輪軸依序穿過「內樑 - 軸承座 - 輪胎 - 軸承座 - 外樑」。

✅ 核心優勢​

  1. 雙面支撐 (Double Support)： 軸的兩端均由軸承固定，受力模式由懸臂式轉變為簡支樑結構，抗彎曲能力提升 4 倍以上。
  2. 抗扭剛性： 兩根 C 型鋼透過鋁柱（Standoffs）鎖定，形成箱體結構（Boxed Structure）。即使側面受到猛烈撞擊，底盤仍能保持幾何精度，不會變形。

3. 工藝細節：軸承座 (Bearing Flats) 與校正​

僅有三明治結構並不足夠，細節處理決定了傳動效率與耐用度。


對齊與同心度 (Alignment)​

兩層樑上的軸承孔位必須完全同心。若有微小歪斜，軸心會產生徑向壓力導致卡死，令馬達過熱（Overheating）。

  • 校正技巧： 先不要鎖緊軸承座的螺絲。將一根長軸同時穿過內外兩側的軸承，利用軸心自動將軸承座校正至同心位置，隨後再鎖緊螺絲。

加固方式 (Reinforcement)​

  • 🚫 禁忌： 嚴禁使用塑膠鉚釘 (Pop Rivets) 固定軸承座。這種零件在激烈碰撞下極易鬆脫。

注意

今個賽季不可使用塑膠鉚釘 (Pop Rivets) 固定軸承座。

  • ✅ 務必： 使用螺絲與尼龍防鬆螺帽 (Nylock Nuts)，將軸承座牢牢鎖定在 C 型鋼上。

🎤 評審面談實戰 (Judges Interview)​

Q1: 為何你們的底盤採用雙層金屬樑結構來固定輪胎？ (考察重點：結構力學與傳動效率)

🧐 回答範例： 「我們採用雙層平行樑結構是為了消除懸臂效應 (Cantilever Effect)。單側支撐在受到撞擊時，輪軸非常容易彎曲變形。透過雙面支撐 (Double-Supported) 設計，輪軸兩端皆由軸承固定，這不僅保護了傳動軸，更提升了底盤整體的抗扭剛性，確保在 Push Back 高強度對抗下，驅動系統仍能保持低摩擦力與結構精度。」

Q2: 我注意到你們的軸承座全部使用螺絲固定，而非塑膠鉚釘，原因為何？ (考察重點：對細節與穩定性的追求)

🧐 回答範例： 「塑膠鉚釘在長時間震動下會產生間隙（Slop），導致軸承產生微小位移，增加不必要的傳動阻力。為了確保幾何精度的穩定性，我們全機使用螺絲與尼龍防鬆螺帽（Nylock Nuts）鎖固。這種『零間隙』的固定方式能保證傳動系統在整場比賽中，即使頻繁碰撞也不會發生軸心偏移。」
