3.3 輪胎配置與防禦策略

前情提要​

在 3.1 與 3.2 章中，我們確認了坦克底盤的優勢，並排除了全向底盤的干擾。現在，我們需要解決一個核心矛盾：如何讓底盤既能順暢轉彎，又能在防守時穩如泰山？答案在於輪胎材質的科學搭配。

1. 輪胎類型分析​

VEX V5 系統中的輪胎主要分為兩大類：全向輪 (Omni Wheels) 與 牽引輪 (Traction Wheels)。

全向輪 (Omni Wheel) —— 機動性的核心​


  • 結構： 輪轂外圍鑲嵌了一圈垂直方向的小滾輪。
  • 特性： 前後滾動時具備正常抓地力；側向滑動時因小滾輪旋轉，摩擦力極低。
  • 優缺點： 優點是轉彎靈活，幾乎沒有轉向阻力；缺點是防禦薄弱。對手從側面施壓時，機器人會因側向零阻力而輕易滑動。

牽引輪 (Traction Wheel) —— 穩固的支點​

  • 結構： 純橡膠輪胎，不具備側向滾輪。
  • 特性： 前後及側向滑動均具備極高的抓地力。
  • 優缺點： 優點是防禦力極佳，難以被橫向推移；缺點是轉彎極其困難。若全機使用牽引輪，轉彎時馬達會因克服巨大的橫向摩擦阻力（Scrub Friction）而過載。

2. 黃金組合：混合配置方案 (Hybrid Configuration)​

為了兼顧兩者優點，主流的設計思維是「鎖死中心」（Anti-shove）的混合配置。

⚙️ 建議配置方案​

六輪坦克底盤 (6WD)：​

  • 前後輪： 使用 全向輪 (Omni Wheel)。確保轉彎時車頭與車尾能順利滑移，減少阻力。
  • 中間輪： 使用 牽引輪 (Traction Wheel)。這對輪胎負責提供核心的側向抓地力，作為底盤的物理錨點。

八輪坦克底盤 (8WD)：​

  • 外側兩排（最前與最後）： 使用 全向輪 (Omni Wheel)。
  • 中間兩排： 使用 牽引輪 (Traction Wheel)。
  • 關鍵注意： 八輪底盤必須配合中心下沉設計 (Drop Center)，否則由於長軸距配合高摩擦橡膠輪，機器人將無法完成原地轉向。

🛡️ 為何此配置最為強大？​

  1. 轉彎時： 配合中心下沉設計，主要接觸點為中間的牽引輪與其中一端的全向輪。全向輪負責引導滑動，而牽引輪提供穩定的旋轉軸心，確保操控靈敏。
  2. 防守時： 當對手嘗試進行 T 型撞擊 (T-Bone) 時，前後全向輪雖無側向阻力，但中間的牽引輪會與地面產生巨大的靜摩擦力。對手必須克服橡膠與地墊間的極限摩擦力才能推動你。

3. 進階防禦：反制「抬升」戰術​

即使具備牽引輪，若對手使用楔形結構（Wedge）將你的機器人一側鏟起，令牽引輪懸空，你的防禦力將瞬間歸零。

⚔️ 解決方案：防鏟側裙 (Side Skirts / Wedges)​


  • 原理： 在底盤兩側安裝金屬板或聚碳酸酯板（Polycarbonate），使其儘可能緊貼地面。
  • 作用： 1. 阻擋對手的鏟子插入車底。 2. 若將側裙設計成斜面，對手撞擊時會被迫「爬」上你的斜坡，這會增加你底盤的向下壓力（Downforce），反而增強了你的抓地力。

🎤 評審面談實戰 (Judges Interview)​

Q1: 為何你們選擇混合使用牽引輪與全向輪？ (考察重點：物理原理與戰術平衡)

🧐 回答範例： 「這是為了在機動性（Agility）與防禦能力（Defense）之間取得平衡。純全向輪底盤雖然轉彎靈活，但側向防禦力不足，極易被對手以 T-Bone 戰術推離得分區。我們透過在中間配置高摩擦力的牽引輪（Traction Wheels），創造了一個側向摩擦錨點，有效防止橫向偏移；同時配合前後全向輪與中置下沉設計（Drop Center），確保機器人仍能保持流暢的轉向性能。」

Q2: 如何防止在對抗中因被抬升而失去抓地力？ (考察重點：防禦性設計細節)

🧐 回答範例： 「我們設計了低重心的防鏟側裙（Anti-Wedge Side Skirts）。底盤兩側的金屬板與地面的間隙被降至最低，以阻擋對手的斜面結構。此外，我們在車尾配置了防傾倒支架（Anti-Tip Bars），確保機器人在劇烈對撞或推擠時，中心驅動輪始終能保持與地面的接觸，防止牽引力失效。」
