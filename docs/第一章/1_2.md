選好了馬達和螺絲（見上一章）後，你手邊已有一堆頂尖零件。但如果你只是將它們隨意鎖在一起，你得到的不是坦克，而是一堆會移動的廢鐵。

在 Push Back 賽季中，賽場如戰場。當兩台重達 7.5 公斤的機械人全速對撞時，結構鬆散的一方會瞬間解體。本章我們將探討三個核心概念：選材、三角支撐及箱體結構。

---

## **材質的抉擇：鋁材 (Aluminum) vs 鋼材 (Steel)**

許多新手會陷入一個迷思：「鋁比鋼貴，鋁比鋼輕，所以全台車都要用鋁材。」錯！在 Push Back 賽季中，**重量也是你的武器**。

### **鋁材：輕巧的飛鳥**
* **特性**：輕、韌性好、價格較高。
* **用途**：**上半部結構 (Upper Structure)**，例如機械臂、吸取裝置、發射器。
* **原理**：上半部結構越輕，重心越低，馬達負載越小，反應速度越快。

:::danger **禁忌**
如果你把底盤做得輕飄飄，對手推你就如推紙箱一樣輕而易舉。
:::

### **🏗 鋼材：穩固的錨**
* **特性**：重、硬度高、價格較便宜。
* **用途**：**底盤（驅動系統／Drivetrain）**，特別是底盤邊框 (Drive Rails) 和配重部分。
* **「反擊」（Push Back）戰術**：在這個賽季，你需要打造一個「不倒翁」。
* **鋼製底盤**：增加抓地力（$F = \mu N$，正向力越大，最大靜摩擦力越大），令對手難以推動你。
* **重心下移**：鋼製底盤 + 鋁製機械臂 = 極低的重心。無論如何碰撞都不容易翻車。

## **幾何學的魔法：三角形(桁架Truss)**

為什麼你的機械人會搖晃不定？因為你使用的是四邊形結構。
* **四邊形**極不穩定，輕輕一推就會變形為平行四邊形。
* **三角形**是物理學中最穩定的形狀。



### **實戰技巧**
* **切勿只鎖一顆螺絲**：兩點確定一條直線，但兩顆螺絲才能固定一個角度。任何連接處至少要使用兩顆螺絲。
* **交叉支撐 (Cross-Bracing)**：如果你有兩座直立的支架 (Towers)，在它們之間加一根斜向的金屬條，形成「X」或「Z」形結構。這能瞬間消除 90% 的晃動。
* **三角加固片 (Gussets)**：在 90 度連接處使用三角形連接片（加固片），而非單純的 L 型片。

---

## **終極硬度：箱體結構 (Boxing)**

普通的 VEX C 型材 (C-Channel) 截面是開口的 `[` 形狀。
* **缺點**：抗扭性 (Torsional stiffness) 極差。你抓住兩端一扭，它就會像麻花一樣變形。
* **缺點**：不耐壓。螺絲鎖得太緊時，開口處會被擠壓變形。

**解決方案：Box It! (將其轉化為箱體結構)**

### 📦 **方法 A：單 C 型材加固**
* **做法**：在 C 型材的凹槽內，每隔一段距離塞入一個鋁柱 (Standoff) 或墊圈 (Spacer)，然後從背面鎖上螺絲固定。
* **作用**：防止斷面被壓扁，適用於機械臂的延伸段。



### 📦 **方法 B：雙 C 型材合併 (C-Channel Boxing)**
* **做法**：拿兩根 C 型材，開口相對扣在一起，中間用鋁柱 (Standoff) 連接。
* **作用**：形成一個封閉的矩形方管。
* **功用**：這是 VEX 中最堅固的結構。底盤兩側安裝車輪的主樑（Drive Rails），**必須**使用這種結構。這能確保輪軸在受到撞擊時，不會因懸臂應力 (Cantilever stress) 而彎曲。
